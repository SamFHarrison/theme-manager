# Code of Conduct

## Our standard

This project expects respectful, constructive collaboration.

Participants should:

- communicate with respect and assume good intent
- focus criticism on ideas, code, and behavior rather than people
- accept feedback in good faith
- avoid harassment, intimidation, discrimination, and personal attacks

## Scope

This applies to project discussions, issues, pull requests, and other community spaces tied to this repository.

## Reporting

If you experience or witness unacceptable behavior, report it through the repository owner via GitHub issue or the contact path documented in `SECURITY.md`.

## Enforcement

The maintainer may remove comments, close discussions, or block contributors who do not follow this code of conduct.
